# Contributing to Simple Addition Project

First off, thank you for taking the time to contribute! This is a learning project, and all types of contributions—whether fixing a typo, optimizing the code, or suggesting new features—are highly appreciated.

## How Can I Contribute?

### 1. Reporting Bugs
If you find a bug or something isn't working quite right:
* Open a new **Issue** on GitHub.
* Describe the problem clearly, including what input caused the error and what you expected to happen.

### 2. Suggesting Enhancements
Want to add a new feature (like handling decimal numbers, adding subtraction, or creating a loop so the program runs multiple times)?
* Open an **Issue** to discuss your idea.
* Clearly explain how the new feature would work.

### 3. Submitting Changes (Pull Requests)
If you want to write the code yourself and submit a change:
1. **Fork** this repository to your own GitHub account.
2. Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3. Make your changes in the code. Ensure your code is clean and properly formatted.
4. **Commit** your changes with a clear message (`git commit -m 'Add support for float data types'`).
5. **Push** to your branch (`git push origin feature/AmazingFeature`).
6. Open a **Pull Request** against the main repository.

## Coding Style Guidelines
* Keep variable names clear and meaningful (e.g., use `num1` and `num2` instead of `a` and `b`).
* Use standard C++ library practices (like using standard streams via `<iostream>`).
* Add comments to explain complex or updated logic.

Thank you for helping make this project better!
